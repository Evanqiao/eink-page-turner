# E-Ink Page Turner

专为安卓墨水屏设备优化的 Obsidian 阅读翻页插件。阅读模式下点击屏幕左侧向上翻页、点击右侧向下翻页，针对墨水屏刷新特性优化体验。

## 功能

- **点击翻页** — 点击屏幕左侧区域向上翻页，右侧区域向下翻页，中间区域保留默认行为
- **防误触** — 自动区分点击与滑动，点击链接、按钮等可交互元素时不触发翻页
- **墨水屏优化** — 禁用平滑滚动和动画效果，消除残影
- **翻页重叠** — 可配置翻页时的内容重叠像素，避免内容断裂
- **全屏支持** — 全屏模式下可选择启用编辑模式翻页

## 安装

### 手动安装

1. 下载 `main.js`、`manifest.json`、`styles.css`
2. 在 Obsidian vault 中创建目录 `.obsidian/plugins/eink-page-turner/`
3. 将三个文件复制到该目录
4. 在 Obsidian 设置中启用插件

### 通过 BRAT 安装

1. 在 Obsidian 社区插件库中搜索 **BRAT** 并启用
2. 打开 BRAT 设置，点击 **Add Beta plugin**
3. 输入仓库地址：`Evanqiao/eink-page-turner`
4. BRAT 会自动下载插件文件并在 Obsidian 中启用

> 每次打开 Obsidian 时 BRAT 会自动检查更新。

## 设置

插件提供以下可配置项：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 左侧点击区域比例 | 25% | 屏幕左侧触发向上翻页的区域 |
| 右侧点击区域比例 | 25% | 屏幕右侧触发向下翻页的区域 |
| 翻页重叠像素 | 50px | 翻页时保留上一页内容的像素量 |
| 编辑模式翻页 | 关闭 | 开启后编辑模式也可使用点击翻页 |

## 兼容性

- 最低 Obsidian 版本：0.15.0
- 支持平台：桌面 + 移动端（安卓墨水屏设备）
- 仅对 Markdown 阅读视图生效
